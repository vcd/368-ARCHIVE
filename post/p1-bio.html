P1: Bio

Create a single webpage containing your biography. Stage (host website live) on GitHub.

  1. Create a repo name p1-bio. Note, please delete previous "p1-about-me".
  2. Create content for your biography using the repo's wiki pages.
  3. Do work. Add content, then style. Refer to TreeHouse. Have fun.
  4. Requirements:
    • Use files from website-starter. Again, do not *clone*, download & create new repo.
    • Use TreeHouse Grid System
    • Use Google Fonts. Choose at least 2(two) fonts.
    • 1(one) image of yourself.
    • Demonstrate good design principles. See TreeHouse.
